Navigation

Besides usual Atom tools for Navigation, API Workbench also provides RAMl-based tools for navigation.

Go To Definition

Putting cursor at a reference to an element and selecting “Go To Definition” action from the context menu will reveal that element in the editor, even if it is located in another file.

Navigating to the following reference types is supported:

  • User-defined types
  • Schemas
  • Resource types
  • Traits
  • Security schemas
  • Included files

Find Usages

As opposed to the “Go To Definition” action, it is possible to find out all the references to a current element.

Putting cursor at a reference to an element and selecting “Go To Definition” action from the context menu will display a popup in the bottom of the editor listing all the places, the element is referenced from. Clicking on an item will reveal that place in the editor.

Outline and Console

There are two other tools, which can also be used for navigation.

Outline displays current file’s contents as a filtered tree and clicking on a node will reveal it in the editor.

Console displays current API resources and methods and clicking on an arrow to the right of the element will reveal it in the editor.
